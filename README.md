## ⚡ About This Project

`smbr` was developed as a **Vibe Coding project** focused on building a reconnaissance and payload assistant designed to speed up initial enumeration and foothold discovery during penetration testing labs, and CTFs.

The tool provides:

- Fast TCP reconnaissance
- Adaptive UDP scanning
- SNMP community enumeration intelligence
- Reverse shell payload generator
- msfvenom payload automation

This project is continuously evolving with new modules such as web enumeration, and attack path suggestion.
